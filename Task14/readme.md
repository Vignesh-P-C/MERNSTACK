## About

A JavaScript program that performs several operations on a given number: sum of first n numbers, multiplication table, sum of digits, Armstrong number check, prime number check, and printing all factors — built to practice loops and conditionals.

## Features

- Sum of the first `n` numbers using a `for` loop
- Multiplication table of `n` using a `for` loop
- Sum of digits of a number using a `while` loop
- Armstrong number check (sum of each digit raised to the power of the digit count)
- Prime number check using a `for` loop up to the square root of the number
- Prints all factors of the number
- Results displayed to the console using `console.log`

## How to Run

1. Download or clone this folder.
2. Keep `index.html`, `style.css`, and `script.js` in the same directory.
3. Open `index.html` in any browser.
4. Open the browser console (F12 → Console tab) to see the logged results.
