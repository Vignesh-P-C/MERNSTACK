# MERN Stack Journey — HTML & CSS Fundamentals

> **Note**
>
> This repository documents my ongoing journey of learning the MERN stack. I maintain a consistent documentation style across every exercise, including beginner-level ones — this is a personal preference for keeping my learning log organized, not an attempt to overstate the complexity of any individual task.

This repo is a collection of small, self-contained HTML & CSS practice tasks, each building on the concepts learned in the previous one — starting from a basic "Hello World" page and progressing through selectors, the box model, layout modes, Flexbox, and responsive design.

Each task lives in its own folder with its own `index.html`, styles, assets, and a dedicated README covering that task's specific goals, features, and what was learned.

## Progression

The tasks are ordered to build layout skills incrementally:

1. **Hello World** – basic HTML5 structure and Flexbox centering.
2. **Laundry Wallah** – a full static page combining multiple content sections.
3. **CSS Selectors** – how specificity and selector types affect styling.
4. **Units, Box Model & Fonts** – viewport-relative sizing and the box model.
5. **Hero Section (Display/Position/Float)** – layout without Flexbox, using `inline-block` and `position`.
6. **Login Page (Flexbox)** – a real two-column layout built with Flexbox.
7. **Responsive (Media Queries)** – taking the Flexbox/inline layouts from earlier tasks and making them responsive across desktop, tablet, and mobile.

## Technologies Used

- HTML5
- CSS3 (Flexbox, positioning, media queries, gradients, viewport units)

## How to Run Any Task

1. Clone this repository.
2. Open the relevant task folder.
3. Open `index.html` in any modern browser (each folder's assets are self-contained, so no build step is required).
