## About

A set of JavaScript functions that each take an array as an argument and perform a single operation on it: finding the maximum number, calculating the sum of all elements, and counting the odd numbers — built to practice different function types and array iteration.

## Features

- `findMax` — a regular function declaration that finds the largest number in the array
- `findSum` — an anonymous function expression that adds up all elements
- `countOdds` — an arrow function that counts how many elements are odd
- A dummy array (`[4, 8, 2, 11, 6, 7, 10]`) used to test all three functions
- Results displayed to the console using `console.log`

## How to Run

1. Download or clone this folder.
2. Keep `index.html`, `style.css`, and `script.js` in the same directory.
3. Open `index.html` in any browser.
4. Open the browser console (F12 → Console tab) to see the logged results.
